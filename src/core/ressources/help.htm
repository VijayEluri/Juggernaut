Juggernaut is a java framework to execute automated builds via commandline.
The builds are defined as Launches which consists of a set of Operations and Triggers.

Juggernaut provides a local history and a HTTP server to browse the results.
Notification mails will be send in case the status of a launch is changing.

Juggernaut user-interface consists of the following tabs:

  • Configuration: Define launches with operations and triggers
  • Scheduler: Control ongoing launches or manually trigger the scheduler
  • Logger: Provides system logging and log-level control
  • History: Browse results in local history
  • Preferences: Define global system preferences
